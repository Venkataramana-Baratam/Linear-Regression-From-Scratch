# 📈 Linear Regression from Scratch

This project implements Linear Regression using only Python and NumPy without any external libraries like scikit-learn.

## Features
- Manual gradient descent
- Loss visualization
- Prediction and evaluation

## Technologies
- Python
- NumPy
- Jupyter Notebook
